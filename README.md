# HTB Academy Scripts & Notes

This repository contains my scripts, notes, and reference material created while completing modules on HTB Academy. Its purpose is to document what I learn through hands-on practice and to build a reusable collection of Bash automation, Linux commands, and cybersecurity techniques.

## Repository Structure

```text
HTB-Academy/
├── modules/
│   ├── <Module Name>/

│   │   ├── cheatsheet.md
│   │   └── scripts/

```

## Contents

Each module may contain:

* Personal notes
* Bash scripts written during the module
* Command references
* Small automation projects
* Cheat sheets for quick review

* Practice Linux system administration
* Build reusable automation scripts
* Reinforce concepts learned through HTB Academy
* Create a personal reference for future penetration testing and red team engagements
